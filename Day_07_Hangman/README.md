# Day 7: Hangman

Welcome to Day 7 of my 100 Days of Code challenge! Today, I combined everything I've learned so far (strings, lists, loops, conditionals, and the random module) to build a fully functional Hangman game.

## Concepts Learned
* Breaking down complex problems into logic steps
* Managing game state (lives, guessed letters)
* Replacing items in a list using the `enumerate()` function
* Working with ASCII art in Python
* Enhancing user experience with clean command-line outputs

## The Project: Hangman
A classic Hangman game! The computer randomly chooses a word from a predefined list, and the player has 6 lives to guess the word, one letter at a time. The game updates the ASCII art of the hangman for every wrong guess.

## How to run the main project
```bash
python main.py
```
