# Day 9: Dictionaries, Nesting and the Secret Auction

Welcome to Day 9 of my 100 Days of Code challenge! Today, I explored Python Dictionaries and Nesting, which are powerful tools for storing structured data.

## Concepts Learned
* Creating and accessing Python Dictionaries (Key-Value pairs)
* Iterating through dictionaries
* Nesting (Lists inside Dictionaries, Dictionaries inside Dictionaries)
* Finding the maximum value in a dictionary using `max()`

## The Project: Secret Auction (FPSBDA)
A command-line program that simulates a First-Price Sealed-Bid Auction. It asks each user for their name and bid, clears the screen so the next user can't see it, and then calculates the highest bidder at the end.
