# Day 8: Function Parameters and Caesar Cipher

Welcome to Day 8 of my 100 Days of Code challenge! Today was all about functions with inputs (arguments and parameters) and using them to build a cryptography program.

## Concepts Learned
* Defining functions with positional and keyword arguments
* Variable scope within functions
* Using the modulo operator (`%`) to wrap around lists/indexes
* String manipulation (replacing characters, lowering case)

## The Project: Caesar Cipher
A classic encryption and decryption algorithm. The user can input a message and a shift number. The program shifts each letter by the given amount through the alphabet, wrapping around from 'z' back to 'a' if necessary.
