# Day 12: Scope & Number Guessing Game

## Project Overview
Day 12 of the "100 Days of Code" challenge dives into the concept of Namespaces and Scope (Local vs. Global variables). The capstone project is a classic Number Guessing Game featuring different difficulty levels that dynamically change the range of the random number.

## Repository Structure
*   **`practice.py`**: Exploring local and global variable scopes, alongside an algorithm to check if a number is prime by counting its divisors.
*   **`project.py`**: The Number Guessing Game logic, utilizing the `random` module, variable assignment based on user input, and tracking player lives.

## Key Concepts Covered
* **Scope:** Understanding how defining variables inside vs. outside functions affects their accessibility.
* **Algorithm Building:** Developing a custom prime number checker using `for` loops and modulo operators.
* **Game Logic:** Using a `while` loop that resolves with an `else` statement to handle the win/loss condition smoothly.

## How to Run
To run the scope and prime checking practice:

    python practice.py

To play the Guessing Game:

    python project.py
